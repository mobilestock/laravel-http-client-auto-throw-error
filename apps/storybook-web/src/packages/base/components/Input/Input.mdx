# Input

O componente **Input** é utilizado para entrada de dados em formulários, suportando diferentes tipos e estados,
como `Password`, `WithError`, entre outros.

# Objetivo

O objetivo deste componente quando usado fora de um formulário é permitir a entrada de dados em um campo de texto,
como um campo de busca, não é necessário informar erros ao usuário.

## Uso Básico

Você pode utilizar o Input com um texto padrão e sem estado de erro.
