# Estilos CSS

Considere os seguintes componentes usando **styled-components**:

```tsx
import styled from 'styled-components'

const ButtonPrimary = styled.button`
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
  font-weight: bold;
`

const ButtonSecondary = styled.button`
  background-color: gray;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
  font-weight: bold;
`

const ButtonDanger = styled.button`
  background-color: red;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
  font-weight: bold;
`
```

Como deve ficar:

```tsx
import styled from 'styled-components'

const BaseButton = styled.button`
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
  font-weight: bold;
`

const ButtonPrimary = styled(BaseButton)`
  background-color: blue;
`

const ButtonSecondary = styled(BaseButton)`
  background-color: gray;
`

const ButtonDanger = styled(BaseButton)`
  background-color: red;
`
```

**Explicação:** Criamos um componente básico `BaseButton` que contém os estilos compartilhados por todos os botões. Em seguida, estendemos esse componente para criar variantes específicas, alterando apenas o `background-color`. Isso elimina a duplicação de código e torna a manutenção dos estilos mais eficiente.
