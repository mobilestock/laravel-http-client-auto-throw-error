# Tipagens

Considere o seguinte código TypeScript:

```ts
function registerUser(user: { name: string; email: string; password: string; }) {
  // Lógica de registro
}

...

function loginUser(user: { email: string; password: string; }) {
  // Lógica de login
}
```

Como deve ficar:

```ts
interface LoginCredentials {
  email: string
  password: string
}
interface User extends LoginCredentials {
  name: string
}

function registerUser(user: User) {
  // Lógica de registro
}

function loginUser(user: LoginCredentials) {
  // Lógica de login
}
```

ou utilizando funções do próprio Typescript:

```ts
interface User {
  name: string;
  email: string;
  password: string;
}

type LoginCredentials = Pick<User, 'email' | 'password'>;

function registerUser(user: User) {
  // Lógica de registro
}

...

function loginUser(user: LoginCredentials) {
  // Lógica de login
}

```

**Explicação:** Definimos uma interface `User` para evitar duplicação na definição do tipo. Utilizamos `Pick` para criar um tipo `LoginCredentials` que reutiliza propriedades de `User`. Isso promove a reutilização de tipagens e mantém o código mais limpo.
