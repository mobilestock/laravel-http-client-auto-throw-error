# Wrappers

Considere os seguintes Providers no app interno e no app entregas `GlobalNavigationContainer` e `GestureHandlerRootView`

```tsx
export default () => {
  return (
    <GlobalNavigationContainer>
      <GestureHandlerRootView>{/*...demais wrappers*/}</GestureHandlerRootView>
    </GlobalNavigationContainer>
  )
}
```

como os dois providers estão sendo referenciados nos apps na mesma forma e sabemos que essa informação não mudará, faz sentido criar um componente central para reutilizar este código.
