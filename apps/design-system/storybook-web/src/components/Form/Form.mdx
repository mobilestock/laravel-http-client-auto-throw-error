# FormInput

O componente **FormInput** é utilizado dentro de formulários para a entrada de dados, integrando-se com bibliotecas de gerenciamento de formulário, como Unform, e suportando diferentes tipos e estados, como `WithError`, entre outros.

## Uso Básico

Você pode utilizar o FormInput dentro de um formulário padrão, sem estado de erro.
