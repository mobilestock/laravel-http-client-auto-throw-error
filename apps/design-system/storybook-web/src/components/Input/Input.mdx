# Input

O componente **Input** é utilizado para entrada de dados em formulários, suportando diferentes tipos e estados, como `Password`, `WithError`, entre outros.

## Uso Básico

Você pode utilizar o Input com um texto padrão e sem estado de erro.

<Controls />

## Estados do Input

### Campo de Senha

Utilize a prop `type="password"` para definir um campo de entrada como senha.

<Canvas of={InputStories.Password} />

### Com Erro

Utilize a prop `error` para indicar um erro na entrada de dados.
