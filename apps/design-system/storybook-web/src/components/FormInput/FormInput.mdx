# FormInput

O componente **FormInput** é utilizado para criar campos de entrada em formulários. Ele oferece suporte a diferentes tipos de entrada, como texto, telefone, e senha, e pode ser configurado para exibir mensagens de erro e ícones de visibilidade para campos de senha.

## Uso Básico

Você pode utilizar o componente `FormInput` para criar um campo de entrada simples com suporte a validação e formatação. Aqui está um exemplo básico de uso:

## Propriedades

- **name**: Nome do campo utilizado pelo Unform para identificação.
- **placeholder**: Placeholder exibido dentro do campo de entrada.
- **type**: Tipo do campo de entrada (`text`, `tel`, `email`, `password`).
- **showErrorMessage**: Exibe mensagem de erro abaixo do campo, se houver.
- **label**: Rótulo exibido acima do campo de entrada.
- **format**: Função opcional para formatar o valor do campo.
